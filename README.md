# AI Notebook

AI Notebook is a local-first Mac desktop application that uses PySide6, Ollama, and eventually RAG to provide a NotebookLM-style experience without depending on cloud services.

## Phase 1 Goal

The first version is intentionally small:

- Load a local `.txt` or `.md` document
- Ask a question about the document
- Send the document and question to a local Ollama model
- Display the model response in the desktop app

## Not in Phase 1

- No cloud sync
- No authentication
- No agents
- No vector database
- No advanced RAG pipeline
- No packaging/distribution

## Tech Stack

- Python
- PySide6
- Ollama
- Local file storage
